What is Devicely

Devicely is a marketplace where users can upload videos of electronic devices and sell them with escrow.

Currently on Rinkeby testnet!

How to sign up?

Navigate to the /signup page and type your username and password. If your username is unique, you can register.

What can I sell here?

Anything as long as it's an electronic device, it can be new, used or completely broken.

How to find electronic devices?

You can search for listed devices after signing up!

How to create a new device listing?

Navigate to My Devices and click on the Add New Device button

How to update a device listing?

After you listed a device you can see it on My Devices, click on the EDIT button

How to use device listing pages?

After you created a device listing, you can access a direct link to it. This link can be shared with anyone. You can see the device listing page on My Devices, click on the LINK button.

How to list a device for free?

If you want to donate the device, just put 0 as price.

How the escrow works?

All listings with a price higher than 0 will need to create an escrow. This means the buyer will need to call a smart contract with metamask. After this is done, the page will redirect to the escrow page. After signing the escrow terms and conditions, the buyer will need to deposit payment into the escrow.

What are the escrow states?
  • Awaiting Payment : It means the buyer didn't deposit anything,yet.
  • Awaiting Delivery : This state means the buyer payed and the seller needs to ship the product
  • Delivered : If the package has been delivered, the buyer needs to change the escrow state to delivered. This means the seller can pull payments.
  • Refunded : If there was a dispute or the seller decided to refund the order, he can do so and in this state the buyer can pull his refund.

How orders work?

Orders are created when somebody clicks the buy button on a device listing page. You can find your orders on the /orders page. Orders can have an escrow or they are free.

What are the order states?
  • Pending : All the orders created are pending. The seller needs to either ship or reject the order.
  • Shipped : The order can be marked shipped.
  • Rejected : The order can be rejected by the seller.

Good to know: The escrow and the order states are separate. When shipping products, make sure you use the two together. Only ship after payment has been deposited!

How does dispute resolution work?

Each order has a dispute button. Press this if your order has has not arrived.

The escrow agent can resolve disputes involving the escrow payments.

  • The escrow can be set to delivered so the seller can pull payments.
  • The escrow can be refunded to the buyer.

This app is not on mainnet, we give no warranty of any kind. Untill we arrive on mainnet, the dispute resolution may not function.
